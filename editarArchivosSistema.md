## 1. Ir a la dirección
```text
rteran@agetic137:/etc/kong$ 
```
## 2. Ver los permisos del archivo
```console
ls -l kong.conf
```
## 3. Dar todos los permisos
```console
sudo chmod 777 kong.conf
```
## 4. Editar el archivo
- Se abrira el editor de texto, editar y guardar
```console
gedit kong.conf
```
