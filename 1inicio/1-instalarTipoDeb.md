# Instalar tipo deb ubuntu
- Ir a la carpeta donde se descargo el archivo
```console
cd Descargas/
```
- Buscar el programa descargado
```console
ls
dia_0.97.2-5_i386.deb//resp
```
- Instalar el paquete
sudo dpkg -i dia_0.97.2-5_i386.deb
